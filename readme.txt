خريد اقساطي با اعتبار قسطا

== Description ==
براحتي امکان خريد اقساطي را به فروشگاه ووکامرس خود اضافه کنيد.

== Installation ==
1. پوشه \'gateway-payping-ghesta-for-woocommerce\' را در مسير \'/wp-content/plugins/\' آپلود کنيد
2. افزونه را از طريق منوي \'افزونه ها\' در وردپرس فعال کنيد
3. تنظيمات افزونه را مي توانيد از طريق قسمت تنظيمات ووکامرس / پيکربندي/ درگاه هاي پرداخت انجام دهيد
4. توکن مورد استفاده در افزونه همان توکن قبلي شما و تنظيم شده در درگاه پي‌پينگ است.
